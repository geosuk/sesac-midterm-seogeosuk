

# CodingOn - Todo List 프로젝트
---
## 프로젝트 소개
이 프로젝트는 기본적인 Todo List를 관리할 수 있는 간단한 프론트엔드 웹 애플리케이션입니다. 사용자는 할 일 목록을 추가하고, 완료 상태를 표시하거나, 목록에서 제거할 수 있습니다.
